# TaskMaster - To Do List App

A simple Android To-Do List application developed using Java and XML in Android Studio.

## Features
- Add Tasks
- Delete Tasks
- Mark Tasks as Completed
- Task Counter
- Completed Task Counter
- Real-Time Clock
- Current Date Display
- Local Storage using SharedPreferences
- Modern UI Design

## Technologies Used
- Java
- Android Studio
- XML
